Сруктура армии взята из игры Варкрафт 3.

Есть 2 фракции Орда и Альянс.
В армии есть 3 типа юнитов:
1) Ближний бой
2) Дальний бой
3) Герои
Есть базовый класс Warrior от него наследуются Infantry, Distance, Hero.
Затем от этих классов наследуются конкретные юниты для фраций.
Используем метод AbstractFactory для создания юнитов.
Есть абстрактная фабрика и от нее наследуются конкретные фабрики
для фракций.

Для создания героев есть возможность добавления предметов
Для этого использован паттерн Builder
Есть абстрактный билдер(HeroBuilder), с помощью которого можно 
герою выдать или забрать оружие или аксессуар(add_item(id)). 
Так как возможность выбора предеметов будет дана игроку в процессе,
конкретых билдеров нет.

Для тестирования использовались gtest
Тесты находятся в tests.cpp
