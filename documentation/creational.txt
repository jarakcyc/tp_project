Сруктура армии взята из игры Варкрафт 3.

Есть 2 фракции Орда и Альянс.
В армии есть 3 типа юнитов:
1) Ближний бой
2) Дальний бой
3) Герой
Армия состоит из множества юнитов 1 и 2 типа и 1 героя.
Есть базовый класс Warrior от него наследуются Infantry, Distance, Hero.
Затем от этих классов наследуются конкретные юниты для фракций.
Используем метод AbstractFactory для создания юнитов.
Есть абстрактная фабрика и от нее наследуются конкретные фабрики
для фракций.

В игре существует 3 типа героев: танк, хил, дамагер.
Для создания героев использован паттерн Builder
В игре есть оружия и аксессуары (на данный момент 2 оружия, 2 аксессура),
у которых есть постоянные и временные(на время боя) эффекты. 
Есть абстрактный билдер(HeroBuilder) с методами add_weapon(), add_accessory().
От него наследуются 3 конкретных билдера: TankBuilder, HealBuilder, DamagerBuilder.
В них соответствующие методы определяются.
Для создания героев есть класс HeroManager, который с помощью определенного 
билдера создает героя.

Для тестирования использовались gtest
Тесты находятся в tests.cpp

Для сборки используется cmake.
